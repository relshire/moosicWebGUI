4. Operation

4.1. Basic player commands

-10

skip back 10 tracks from history

prev

skip back one track

play

play next track from playlist track

stop

stop playing current track (if any)

pause

toggle pause mode on/off

next

skip forward one track

+10

skip forward 10 tracks

remove

remove current track from playlist; this option is available only in looping mode

4.2. Player modes

Looping

when loop mode is on, songs are returned to the end of the song queue when they finish playing instead of being discarded

Advancing

when advancing mode is on new songs are going to be played from the queue after the current song is finished

Pause

when pause mode is on the currently playing song is suspended

4.3. View modes

Normal

in normal view mode the main content box shows the 10 upcoming tracks from the playlist as well as the last 10 tracks from the history

Browse playlist

displays at most 250 tracks from the playlist, or, if a search pattern is specified, all tracks from the playlist matching the pattern

Browse history

displays at most 250 tracks from the history buffer, or, if a search pattern is specified, all tracks from the history buffer matching the pattern

Browse local files

displays all files and music tracks in a directory in the jukebox subtree

Tree view

displays all directory manes of the jukebox subtree

4.4. General menu

Normal view

switch to normal view mode

Browse playlist

switch to browse playlist mode

Browse history

switch to browse history mode

Browse local files

switch to browse local files mode

Tree view

switch to tree view mode

Re-scan jukebox dir

re-read all files from jukebox subtree

Exit

exit moosicWebGUI (requires confirmation)

4.5. Playlist menu

Shuffle

rearrange the contents of the playlist into a random order

Sort

rearrange the contents of the playlist into sorted order

Reverse

rearrange the contents of the playlist into reverse order

Delete duplicates

remove all duplicate entries from the playlist

Clear

remove all entries from the playlist (requires confirmation)

Save current playlist

prepare a page for saving the playlist's content to a file (in ,m3u format)

Load playlist from file

load the content of a playlist file (in .m3u format) into the playlist; load options: replace playlist, append to end, prepend to top or mix-in.

4.6. Memory menu

Add

add current track to special memory playlist

Show

show content of special memory playlist

Clear

remove all tracks from special memory playlist (requires confirmation)
